# Jenkins CI/CD Pipeline on AWS EC2

Automated multi-environment CI/CD pipeline using Jenkins on AWS EC2.

## Pipeline Stages
1. Code Checkout — pulls code from GitHub
2. Build — compiles application
3. Test — runs automated tests
4. Deploy SIT — deploys to SIT environment
5. Deploy UAT — deploys to UAT environment
6. Deploy Production — deploys with manual approval

## Tech Stack
- Jenkins
- AWS EC2 (Ubuntu)
- GitHub
- Bash Scripts
